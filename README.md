# Rating-Component
 
-Este proyecto es una implementación del desafío "Interactive Rating Component" de Frontend Mentor. El objetivo es crear un componente de calificación interactivo que permita a los usuarios seleccionar una calificación y enviar su opinión.

1. git clone [URL del repositorio]

2. cd interactive-rating-component
Navega al directorio del proyecto.

*Tecnologías Utilizadas, React, typeScript* 
- El componente de calificación interactivo permite a los usuarios seleccionar una calificación de 1 a 5. Una vez seleccionada, el usuario puede enviar su calificación, y el componente mostrará un mensaje de agradecimiento.
- Integra el componente en tu aplicación web copiando el código HTML y CSS proporcionado. Asegúrate de incluir cualquier script necesario para la funcionalidad interactiva.
